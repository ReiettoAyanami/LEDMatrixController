# LED Matrix controller

A simple application built in python to control an 8*8 LED matrix.

## Specs:

LED Type:   WS2811

Python version: 3.10.1

Pygame Version: 2.1.0



## What i'm using:

- ELEGOO NANO

- CJMCU 8*8 LED Matrix

- Random breadboard
